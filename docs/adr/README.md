# Architecture Decision Records (ADRs)

This folder contains architecture and implementation decisions for the Workspace App.

## ADR Index

- ADR-0001: V1 Technology Stack (.NET 8 + C# + WPF)
- ADR-0002: Explicit Window State in V1 Capture/Restore

## Conventions

- File naming: `NNNN-short-kebab-case-title.md`
- Status values: `Proposed`, `Accepted`, `Superseded`, `Deprecated`
- Do not rewrite history in accepted ADRs.
- If a decision changes, create a new ADR and reference the old one as superseded.
